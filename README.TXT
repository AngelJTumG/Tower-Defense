# 🏰 Tower Defense

Juego desarrollado en **Java con Greenfoot** inspirado en el género **Tower Defense**.

El objetivo principal es defender el castillo de oleadas de zombies utilizando torretas colocadas estratégicamente en el mapa.

---

# 🎮 Descripción del juego

En este juego los zombies recorren un camino fijo intentando llegar al castillo.

El jugador debe colocar torretas en las posiciones marcadas con una **X rosada** para detenerlos antes de que destruyan la base.

Cada zombie que logra llegar al castillo reduce una vida.
